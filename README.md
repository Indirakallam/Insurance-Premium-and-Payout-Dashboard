Project Title:
Insurance Premium & Payout KPI Dashboard

Short Description & Purpose:
A Power BI dashboard designed to analyze insurance premiums, claims, payouts, and loss ratios.
The purpose of this project is to demonstrate how raw insurance data can be transformed into business‑ready insights that support underwriting, risk assessment, and financial decision‑making.

Tech Stack:
Power BI Desktop – Data modeling, DAX, interactive reporting
Power Query – Data cleaning & transformation
Excel/CSV Dataset – Insurance premium & claims data
Business Analysis Techniques – KPI definition, domain understanding, dashboard storytelling

Data Source:
Publicly available Insurance Premium & Claims dataset (CSV/Excel) containing:
Customer demographics
Policy details
Premium amounts
Claim amounts
Payout status
Risk categories

Business Impact:
This dashboard enables insurance teams to:
Monitor premium collection and claim payouts in real time
Identify high‑risk segments and loss‑making policy categories
Improve underwriting decisions through data‑driven insights
Detect rising claim trends early to reduce financial exposure
Support executive reporting with clear KPIs and visual storytelling
